# Frontend BichoFull

Aplicação Angular responsável pela interface do sistema.

## Rodando localmente

```bash
npm install
npm start
```

A aplicação ficará disponível em `http://localhost:4200`.

## Rodando com Docker

A partir da raiz do projeto:

```bash
docker compose up --build frontend
```

## Scripts

```bash
npm start
npm run build
npm test
```

## Observação

Durante o desenvolvimento, o frontend consome a API em `http://localhost:8080`.
